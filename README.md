🧩 Problems Solved
✅ Problem 1: Mobile Recharge Validation System

Validates recharge amount against valid plans (199, 299, 399, 599)
Minimum recharge must be ₹50
Allows retry using while loop

Concepts Used: Function, while loop, List

✅ Problem 2: Inventory Reorder Alert System

Checks stock levels for each product
Prints Reorder Alert if stock < 15, else Stock OK

Concepts Used: Function, for loop, Dictionary

✅ Problem 3: Student Result Processing System

Accepts list of student marks
Calculates average and returns Pass or Fail

Concepts Used: Function, for loop, List

✅ Problem 4: Cab Fare Estimator with Retry Option

Calculates fare with base fare + per km charge
Applies 25% extra for peak hours
Allows retry using while loop

Concepts Used: Function, while loop, Conditional Logic

✅ Problem 5: Employee Attendance Eligibility Checker

Takes day-by-day P/A input from user
Calculates attendance percentage
Returns Eligible if ≥ 75% else Not Eligible
Handles ZeroDivisionError

Concepts Used: Function, for loop, Input Validation

✅ Problem 6: Password Strength Checker

Checks minimum length, digit, and special character
Returns Strong or Weak password result

Concepts Used: Function, for loop, String Traversal
